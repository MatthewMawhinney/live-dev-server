Static-Live-Server

Static-Live-Server (SLS) is a project I created to learn and compare npm scripts, gulp, and webpack as task runners and build tools.

Why

SLS was created to optimize the creation of static sites, SLS will watch changes to your HTML, CSS, SASS, JS, and Images and reload the browser accordingly.

What

Using BrowserSync, SLS allows you to compile your site to a production build, test that build across multiple devices over your network, and test dependencies that require HTTPS.

How

SLS will compile SASS to CSS, concatenate and minify CSS, concatenate and minify JS, compile ES6 to ES5, and minify images.

Commands to run each tool...

NPM Scripts:

npm run dev-server

Gulp.js:

gulp

Webpack:

npm run webpack:dev

or

npm run webpack:build

Parcel:

npm run parcel
